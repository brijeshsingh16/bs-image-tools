# 🚀 BS Image Tools

<p align="center">

A modern, fast, and privacy-focused **browser-based image toolkit** that allows users to compress images, convert image formats, and create PDF files directly from the browser.

**No uploads. No servers. 100% client-side processing.**

# 📌 About The Project

**BS Image Tools** is a production-ready image utility web application built using pure frontend technologies.

The goal of this project is to provide simple, fast, and secure image processing tools without requiring users to upload their files to any external server.

All operations are performed directly inside the user's browser.

---

# ✨ Features

## 🖼️ Image Compressor

* Compress images instantly in the browser
* Reduce image file size while maintaining quality
* Before & After image preview
* Download compressed images
* Fast client-side processing

---

## 🔄 Image Converter

Convert images between different formats.

### Supported Formats:

* JPG
* PNG
* WEBP

Features:

* Instant conversion
* No server processing
* Browser-based image handling

---

## 📄 Image To PDF Converter

Create PDF documents from images.

Features:

* Convert multiple images into one PDF
* Generate PDF instantly
* Download generated PDF file
* Powered by jsPDF
